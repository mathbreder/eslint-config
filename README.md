# mathbreder's ESLint config

## Whats included?

- Standard config base;
- Prettier;

## Setup

### Node.js

Install dependencies:
```
npm i -D eslint https://github.com/mathbreder/eslint-config.git
```
Inside `.eslintrc.json`
```
{
  "extends": "@mbreder/eslint-config/node"
}
```
